# Installation

### Cloner le dépôt :

 ```bash
git clone https://github.com/yelineeee/StationMeteo.git
```


### Se placer dans le dossier du projet et installer les dépendances :
 ```bash
cd StationMeteo
npm i
```

### Modifier les information de connexions dans esp32/station.ino 

```
const char* ssid = "exemple_Wifi";
const char* password = "jeSuisUnMotDePasseTresFort123!";
```

### Lancer le bridge

 ```bash
cd bridge-temp
node index.js
```
